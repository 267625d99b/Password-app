# 🔐 مدير كلمات المرور

تطبيق ويب آمن لإدارة كلمات المرور والحسابات بتصميم Google Material Design.

## ✨ المميزات

- 🔒 **تشفير AES-256-GCM** - أعلى درجات الأمان
- 🎨 **تصميم جميل** - واجهة مستخدم حديثة وسهلة الاستخدام
- 📱 **PWA** - يعمل كتطبيق على الهاتف والويب
- 🔄 **مزامنة البيانات** - نسخ احتياطي واستيراد سهل
- 🌙 **الوضع الداكن** - دعم المظهر الفاتح والداكن
- 🔎 **البحث السريع** - ابحث عن حساباتك بسهولة
- 📂 **التصنيفات** - نظم حساباتك بالفئات
- 🔐 **كلمة مرور رئيسية** - حماية كاملة لبياناتك

## 🚀 البدء السريع

### المتطلبات
- Node.js 14+
- npm أو yarn

### التثبيت المحلي

```bash
# استنساخ المستودع
git clone https://github.com/267625d99b/Password-app.git
cd Password-app

# تثبيت التبعيات
npm install

# تشغيل التطبيق محلياً
npm start
# أو للتطوير
npm run dev

# الآن افتح المتصفح على:
# http://localhost:3001
```

## 📖 الاستخدام

1. **إنشاء كلمة مرور رئيسية** - أدخل كلمة مرور قوية عند أول فتح للتطبيق
2. **إضافة حساب** - اضغط زر الإضافة (+) وأدخل بيانات الحساب
3. **البحث والتصفية** - ابحث عن حساباتك أو صنفها حسب النوع
4. **النسخ الاحتياطي** - احفظ نسختك الاحتياطية من الإعدادات
5. **الاستيراد** - استعد بياناتك من نسخة احتياطية سابقة

## 🔐 الأمان

- **التشفير من طرف المستخدم**: جميع البيانات مشفرة بـ AES-256-GCM
- **عدم حفظ كلمات المرور**: كلمة المرور الرئيسية لا تُحفظ في الخادم
- **PBKDF2**: استخدام دالة اشتقاق قوية للمفاتيح
- **HTTPS مطلوب**: في الإنتاج، استخدم HTTPS دائماً
- **بيانات محلية**: البيانات تُخزن بشكل محلي وآمن


## 🛠️ تطوير

### البنية الخلفية (Backend)
- **Express.js** - خادم ويب بسيط وسريع
- **Node Crypto** - تشفير آمن من Node.js
- **PBKDF2** - اشتقاق مفاتيح آمن

### البنية الأمامية (Frontend)
- **HTML5** - معايير حديثة
- **CSS3** - تصميم جميل وسريع الاستجابة
- **Vanilla JavaScript** - بدون إطارات عمل، أسرع وأخف
